ชุดประจำชาติของอาเซียน

ชุดประจำชาติ ประเทศอาเซียนมีชุดแต่งกายประจำชาติที่มีความแตกต่างกันตามวัฒนธรรมของตนเอง แต่สิ่งที่เหมือนกัน คือ เป็นชุด

แต่งกายที่สุภาพมิดชิดเรียบร้อย

1.บรูไนดารุสซาลาม                              

ผู้ชาย : เรียกว่า บลาจู มลายู คือ สวมเสื้อแขนยาวนุ่งโสร่งทับกางเกงขายาว                                          

ผู้หญิง : เรียกว่า บาจู กูรุง คือ สวมเสื้อคลุมยาว กางเกงหรือกระโปรงยาว สวมผ้าคลุมศรีษะ                                             

2.กัมพูชา

ผู้ชาย : สวมเสื้อคอปิด นุ่งโจงกระเบน

ผู้หญิง : สวมเสื้อแขนยาว นุ่งผ้าถุง คาดเข็มขัดทับ

3.อินโดนีเซีย

ผู้ชาย : สวมเสื้อแขนยาว นุ่งกางเกงขายาว เรียกว่า กาอิน

ผู้หญิง : สวมเสื้อพอดีตัวแขนยาว เรียกว่า เคบาย่าและนุ่งโสร่งที่มีสีสัน

4.ลาว

ผู้ชาย : มีเอกลักษณ์ที่การผูกผ้าขาวม้าที่เอว นุ่งโจงกระเบน สวมเสื้อคอกลม

ผู้หญิง : นุ่งซิ่น สวมเสื้อแขนกระบอก เกล้าผม ห่มสไบเฉียง

5.มาเลเซีย

ผู้ชาย : มีเอกลักษณ์ที่การสวมหมวก นุ่งโสร่งทับกางเกงขายาว สวมเสื้อแขนยาว เรียกว่า บาจู มลายู

ผู้หญิง : สวมเสื้อคลุมยาวและกางเกงหรือกระโปรงยาว สวมผ้าคลุมศรีษะและผ้าคล้องคอ เรียกว่า บาจู กูรุง

6. เมียนมา

ผู้ชาย : สวมเสื้อแขนยาว และนุ่งโสร่งที่เรียกว่า ลองยี

ผู้หญิง : สวมเสื้อแขนกระบอก และนุ่งโสร่งที่เรียกว่า ลองยี

7. ฟิลิปปินส์

ผู้ชาย : สวมเสื้อแขนยาว มีปกทำจากผ้าใยสัปปะรด เรียกว่า บารองตากาล็อก

ผู้หญิง : สวมกระโปรง สวมเสื้อแขนสั้นจับจีบยกตั้งเหนืไหล่ เรียกว่า บาลินตาวัก

8. สิงคโปร์

ผู้ชาย : แต่งกายด้วยชุดสากล แบบตะวันตก

ผู้หญิง : สวมเสื้อตัวยาวคลุมสะโพก มีผ้าคล้องคอและนุ่งกระโปรงยาว

9. ไทย

ผู้ชาย : : แต่งกายด้วยชุดไทยพระราชทาน

ผู้หญิง : แต่งกายด้วยชุดไทยจักรี ประกอบด้วยสไบเฉียง

และผ้าซิ่น มีจีบยกข้างหน้า

10.เวียดนาม

ผู้ชาย : สวมชุดยาวพอดีตัว นุ่งกางเกงขายาวไว้ด้านใน

ผู้หญิง : สวมชุดอ่าวหญ่าย เป็นชุดผ้ายาวพอดีตัวนุ่งกางเกงขายาวไว้ด้านใน
