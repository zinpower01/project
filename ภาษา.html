ภาษาของประเทศในอาเซียน

ภาษา  ในอาเซียนมีการชกำหนดภาษาเป็นภาษาราชการของตนเองขึ้นใช้ภายในประเทศ นอกจากนี้ยังมีภาษาอื่นๆปะปนอยู่บ้างแต่ละ

ประเทศมีการใช้ภาษาต่อไปนี้

ประเทศ : บรูไนดารุสซาลาม

ภาษราชการ : บาฮาซามาเลเซีย

ภาษาอื่นๆ : จีน อังกฤษ

ประเทศ : เมียนมา

ภาษราชการ : ภาษาพม่า

ภาษาอื่นๆ : อังกฤษ มอญ ยาวี

ประเทศ : กัมพูชา

ภาษราชการ : เขมร

ภาษาอื่นๆ : เวียดนาม ฝรั่งเศส

ประเทศ : ฟิลิปปินส์

ภาษราชการ : ตากาล็อกและอังกฤษ

ภาษาอื่นๆ : สเปน จีน ทมิฬ มาเลเซีย

ประเทศ : อินโดนีเซีย

ภาษราชการ : บาฮาซาอินโดนีเซีย

ภาษาอื่นๆ : จีน ดัตซ์

ประเทศ : สิงคโปร์

ภาษราชการ : อังกฤษ

ภาษาอื่นๆ : จีน มาเลเซีย อินโดนีเซีย

ประเทศ : ลาว

ภาษราชการ : ภาษาลาว

ภาษาอื่นๆ : ไทย ฝรั่งเศส ม้ง อังกฤษ ไท

ประเทศ : ไทย

ภาษราชการ : ภาษาไทย

ภาษาอื่นๆ : อังกฤษ จีน ท้องถิ่น

ประเทศ : มาเลเซีย

ภาษราชการ : บาฮาซามาเลเซีย

ภาษาอื่นๆ : จีน อังกฤษ ยาวี

ประเทศ : เวียดนาม

ภาษราชการ : เวียดนาม

ภาษาอื่นๆ : จีน ฝรั่งเศษ เขมร
